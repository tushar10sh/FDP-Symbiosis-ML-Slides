Introduction to Neural Networks with Keras

Building Blocks

  • Neuron (Perceptron): Computes weighted sum of inputs + bias.
  • Activation Functions: Adds non-linearity.
    • ReLU: max(0, z) (Hidden layers default)
    • Sigmoid/Softmax: Probabilities (Output layer)
Multilayer Perceptron (MLP): Input Layer → Hidden Layers → Output Layer

Keras Sequential API

# 1. Build the Model
model = keras.models.Sequential([
    keras.layers.Dense(300, activation="relu", input_shape=[28*28]),
    keras.layers.Dense(100, activation="relu"),
    keras.layers.Dense(10, activation="softmax") # Output: 10 classes
])

# 2. Compile (Loss, Optimizer, Metrics)
model.compile(
    loss="sparse_categorical_crossentropy",
    optimizer="sgd",
    metrics=["accuracy"]
)

# 3. Fit (Train)
history = model.fit(
    X_train, y_train,
    epochs=30,
    validation_data=(X_valid, y_valid)
)
