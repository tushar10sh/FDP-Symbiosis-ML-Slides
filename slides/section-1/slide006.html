Supervised Learning

Slide 6 / 15

Core Idea

The training set you feed to the algorithm includes the desired solutions, called labels. The model learns to map inputs → outputs.

Task 1 — Classification

Predict a discrete class label for each input. The spam filter is a classic binary classification example (spam / ham).

Key concepts:

  • Binary classification: Two classes (yes / no)
  • Multiclass classification: Many classes (digit 0–9)
  • Attributes vs. Features: Attribute = data type; Feature = attribute + value
  • Labels: The "right answers" in training data

Example: Spam Filter

Input: email text & metadata → Output: SPAM or HAM

Task 2 — Regression

Predict a continuous numeric value (target). Requires giving the algorithm many examples including both predictors and the target value.

Key concepts:

  • Predictors: Input features (mileage, age, brand…)
  • Target: Output value to predict (e.g., car price)
  • Note: Some regression algorithms can also classify, and vice versa (e.g., Logistic Regression)

Example: Car Price Prediction

Input: mileage, age, brand → Output: price in USD
